# Django Starter Project

A clean Django project scaffold demonstrating the core project structure with URL routing and settings configuration.

## Overview
Minimal Django project setup for learning the framework's entry points: manage.py, URL dispatcher, WSGI/ASGI handlers, and project settings.

## Technologies Used
- **Framework:** Django
- **Language:** Python 3

## Project Structure
```
manage.py
intro/
+-- settings.py
+-- urls.py
+-- wsgi.py
+-- asgi.py
```

## Getting Started
```bash
pip install django
python manage.py runserver
```
